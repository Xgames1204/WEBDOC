HTML Теги и Атрибуты

Тег Атрибуты Описание
<html> lang, dir, xmlns, manifest Основной контейнер документа. Определяет язык, направление и манифест.
<head> — Содержит метаданные документа (например, стили, шрифты, скрипты).
<meta> name, content, charset, http-equiv, property, http-equiv, content-type, viewport Определяет метаданные документа: описание, кодировка, ключевые слова и настройки для мобильных устройств.
<title> — Заголовок страницы, отображаемый на вкладке браузера.
<base> href, target Определяет базовый URL для всех ссылок в документе.
<link> rel, href, type, sizes, media, crossorigin, integrity, hreflang, referrerpolicy Подключает внешние ресурсы, например, стили или шрифты.
<style> type, media, scoped Встраивает CSS-стили в документ.
<body> onload, onunload, class, id, style Содержит основной контент страницы.
<h1>, <h2>, <h3>, <h4>, <h5>, <h6> align, class, id, style Заголовки разных уровней (от самого важного к менее важным).
<p> align, class, id, style Определяет абзац текста.
<a> href, target, rel, title, download, hreflang, ping, referrerpolicy, type Создает ссылку на другой документ или ресурс.
<img> src, alt, title, width, height, loading, usemap, ismap, decoding Вставляет изображение, поддерживает различные атрибуты для размеров и загрузки.
<div> id, class, style, data-*, lang, role, aria-* Контейнер для других элементов, часто используется для структуры.
<span> id, class, style, data-* Строчный контейнер для текста или других элементов.
<ul> type, compact, id, class Создает ненумерованный список.
<ol> type, start, reversed, id, class Создает нумерованный список.
<li> value, id, class, style Элемент списка.
<form> action, method, name, target, enctype, autocomplete, accept-charset, novalidate Форма для ввода данных пользователя, поддерживает различные атрибуты для отправки данных.
<input> type, name, value, placeholder, required, checked, disabled, readonly, maxlength, min, max, step, autofocus, accept, pattern, multiple Поле ввода для различных типов данных (текст, пароль, чекбокс и т. д.).
<button> type, name, value, disabled, onclick, autofocus, formaction, formmethod Кнопка, используемая для отправки формы или выполнения действий.
<textarea> rows, cols, placeholder, name, required, maxlength, readonly, autofocus Многострочное поле ввода текста.
<select> name, multiple, size, required, disabled, autofocus, form, autofocus Выпадающий список для выбора значений.
<option> value, selected, disabled, label, class, id Элемент списка в <select>.
<optgroup> label, disabled, class, id Группирует элементы <option> в выпадающем списке.
<label> for, id, class, style Связывает текст с элементом формы (например, с полем ввода).
<table> border, cellspacing, cellpadding, width, height, align, summary, caption, role Создает таблицу.
<tr> align, valign, class, id Определяет строку таблицы.
<td> colspan, rowspan, align, valign, width, height, style, data-* Ячейка таблицы.
<th> colspan, rowspan, scope, align, valign, style, data-* Заголовочная ячейка таблицы.
<thead> — Группирует заголовки таблицы.
<tbody> — Группирует содержимое таблицы.
<tfoot> — Группирует подвал таблицы.
<iframe> src, width, height, name, frameborder, scrolling, allowfullscreen, sandbox, allow, referrerpolicy Встраивает другой HTML-документ в текущий.
<object> data, type, width, height, name, usemap, classid, codebase Встраивает объект, например, Flash, PDF и т. д.
<embed> src, type, width, height, autoplay, loop, volume, muted Встраивает внешний мультимедийный контент, например, аудио или видео.
<video> src, controls, autoplay, loop, muted, poster, width, height, preload, playsinline, autoplay Вставляет видео.
<audio> src, controls, autoplay, loop, muted, preload, volume, autoplay, type Вставляет аудиофайл.
<canvas> width, height, id, class, style, data-* Создает область для рисования, используется с JavaScript.
<svg> width, height, viewBox, version, xmlns, id, class, style Вставляет SVG-графику.
<math> xmlns, xmlns:xlink, id, style Встраивает математические выражения в формате MathML.
<noscript> — Содержит текст, который отображается, если JavaScript отключен в браузере.
<ruby> xmlns, class, style Вставляет аннотированный текст, обычно используется для китайских иероглифов или транскрипций.
<rt> — Определяет транскрипцию в контексте тега <ruby>.
<rp> — Предоставляет альтернативное представление для браузеров без поддержки <ruby>.
<wbr> — Разрешает браузеру разрывать строку в заданной точке.
<b> — Делает текст жирным.
<i> — Делает текст курсивным.
<u> — Подчеркивает текст.
<strong> — Выделяет текст как важный, обычно жирным.
<em> — Выделяет текст как акцентированный, обычно курсивом.
<mark> — Выделяет текст, обычно желтым фоном.
<small> — Делает текст меньшим.
<del> — Перечеркнутый текст.
<ins> — Текст, который был вставлен в документ.
<q> — Вставляет короткую цитату.
<blockquote> — Вставляет длинную цитату.
<cite> — Ссылается на источник цитаты.
<code> — Отображает программный код.
<pre> — Сохраняет форматирование текста (например, пробелы, переводы строк).
<sub> — Нижний индекс.
<sup> — Верхний индекс.
<bdi> — Определяет область для изолирования текста с измененным направлением.
<bdo> dir Изменяет направление текста (например, справа налево).
<ruby> — Представляет текст с аннотациями (обычно для китайских иероглифов).

Свойства CSS

Свойство Описание Пример
accent-color Устанавливает основной цвет для элементов управления (например, для кнопок и флажков). accent-color: blue;
align-content Определяет выравнивание содержимого в контейнере с флексбоксом или сеткой (по поперечной оси). align-content: center;
align-items Устанавливает выравнивание флекс-элементов внутри контейнера по поперечной оси. align-items: flex-start;
align-self Устанавливает индивидуальное выравнивание элемента по поперечной оси в контейнере с флексбоксом. align-self: stretch;
all Сбрасывает все свойства, установленные для элемента, к их начальным значениям. all: unset;
animation-delay Устанавливает задержку перед началом анимации. animation-delay: 1s;
animation-duration Устанавливает продолжительность анимации. animation-duration: 3s;
animation-iteration-count Определяет количество повторений анимации. animation-iteration-count: infinite;
animation-timing-function Устанавливает временную функцию анимации, определяющую её скорость. animation-timing-function: ease-in-out;
backface-visibility Устанавливает, видима ли задняя сторона элемента при поворотах. backface-visibility: hidden;
background-blend-mode Устанавливает режим наложения для фона, если его несколько. background-blend-mode: multiply;
background-position-x Устанавливает горизонтальное положение фонового изображения. background-position-x: center;
background-position-y Устанавливает вертикальное положение фонового изображения. background-position-y: top;
border-collapse Устанавливает, будут ли рамки таблицы объединяться или разделяться. border-collapse: collapse;
border-image Устанавливает изображение в качестве рамки. border-image: url('border.png') 30 stretch;
border-image-outset Устанавливает, на сколько элементов рамка выходит за пределы элемента. border-image-outset: 5px;
border-image-repeat Определяет, как будет повторяться изображение рамки. border-image-repeat: round;
border-image-slice Определяет, как изображение рамки будет нарезано. border-image-slice: 30%;
border-left Устанавливает свойства для левой рамки элемента. border-left: 3px solid red;
border-right Устанавливает свойства для правой рамки элемента. border-right: 3px solid blue;
border-top Устанавливает свойства для верхней рамки элемента. border-top: 3px solid green;
border-bottom Устанавливает свойства для нижней рамки элемента. border-bottom: 3px solid yellow;
box-decoration-break Устанавливает, как должны обрабатываться разрывы в строках для фона и рамок. box-decoration-break: clone;
box-shadow Устанавливает тень для элемента. box-shadow: 10px 10px 5px rgba(0, 0, 0, 0.5);
break-before Устанавливает, должен ли элемент начинаться с новой строки (для разбиения текста). break-before: page;
break-after Устанавливает, должен ли элемент завершаться с новой строки. break-after: column;
break-inside Определяет, где можно разорвать элемент внутри контейнера. break-inside: avoid;
caret-color Устанавливает цвет курсора в текстовых полях. caret-color: red;
clip-path Обрезает элемент по заданной форме (например, круг). clip-path: ellipse(50% 50% at 50% 50%);
column-count Устанавливает количество колонок в многостолбцовом макете. column-count: 3;
column-fill Устанавливает, как должны заполняться колонки (высота колонок). column-fill: balance;
column-gap Устанавливает зазор между колонками в многостолбцовом макете. column-gap: 20px;
column-rule Устанавливает стиль, цвет и толщину линии между колонками. column-rule: 1px solid black;
column-span Устанавливает, сколько колонок должен занимать элемент в многостолбцовом макете. column-span: all;
contain Устанавливает, как должны быть ограничены элементы внутри контейнера. contain: layout;
content Устанавливает содержимое, которое будет вставлено перед или после элемента (например, для ::before). content: "New text";
counter-increment Увеличивает счетчик, используемый для генерации числовых значений. counter-increment: section;
counter-reset Сбрасывает счетчик на начало. counter-reset: section 0;
direction Устанавливает направление текста (например, слева направо или справа налево). direction: rtl;
display Устанавливает, как элемент должен отображаться (например, block, inline). display: block;
empty-cells Устанавливает, должны ли отображаться пустые ячейки таблицы. empty-cells: show;
filter Применяет графические фильтры, такие как размытие и контрастность. filter: grayscale(100%);
flex-basis Устанавливает начальный размер флекс-элемента до распределения свободного пространства. flex-basis: 200px;
flex-grow Устанавливает, насколько флекс-элемент будет расти относительно других элементов. flex-grow: 2;
flex-shrink Устанавливает, насколько флекс-элемент будет сжиматься относительно других элементов. flex-shrink: 1;
flex-wrap Устанавливает, как флекс-элементы должны оборачиваться в контейнере. flex-wrap: wrap-reverse;
gap Устанавливает зазоры между элементами в сетке или флекс-контейнере. gap: 10px;
grid-template-columns Определяет структуру столбцов сетки (Grid). grid-template-columns: 1fr 2fr;
grid-template-rows Определяет структуру строк сетки (Grid). grid-template-rows: 100px 200px;
grid-gap Устанавливает зазоры между строками и колонками в сетке (Grid). grid-gap: 10px;
grid-template-areas Устанавливает именованные области в сетке (Grid). grid-template-areas: 'header header' 'main sidebar';
justify-content Выравнивает элементы по основной оси (для флексбоксов и сетки). justify-content: space-between;
justify-items Выравнивает элементы внутри ячеек сетки (Grid). justify-items: center;
justify-self Выравнивает элемент внутри своей ячейки в сетке (Grid). justify-self: end;
left Устанавливает левый отступ для позиционированного элемента. left: 10px;
letter-spacing Устанавливает расстояние между буквами текста. letter-spacing: 2px;
line-height Устанавливает высоту строки текста. line-height: 1.5;
list-style Устанавливает стиль маркера для списков. list-style: square inside;
margin Устанавливает внешние отступы для элемента. margin: 20px 10px 30px 5px;
margin-bottom Устанавливает внешний отступ снизу для элемента. margin-bottom: 20px;
margin-left Устанавливает внешний отступ слева для элемента. margin-left: 15px;
margin-right Устанавливает внешний отступ справа для элемента. margin-right: 15px;
margin-top Устанавливает внешний отступ сверху для элемента. margin-top: 10px;

JS

Тема/Свойство/Метод Описание Пример Передаваемые данные
Объекты
Object Базовый объект, от которого наследуются все другие объекты в JavaScript. let obj = new Object(); Нет
Object.create() Создает новый объект с указанным прототипом. let obj = Object.create(protoObj); Прототип (объект)
Object.freeze() Замораживает объект (делает его неизменным). Object.freeze(obj); Объект (для заморозки)
Object.seal() Закрывает объект от добавления новых свойств. Object.seal(obj); Объект (для запечатывания)
Object.is() Сравнивает два значения на строгое равенство (включая NaN). Object.is(0, -0) → false Два значения (для сравнения)
Object.assign() Копирует значения всех собственных перечисляемых свойств из одного или нескольких исходных объектов в целевой объект. Object.assign(target, source); Целевой объект и исходные объекты
Object.values() Возвращает массив собственных перечисляемых значений объекта. Object.values(obj); Объект (для извлечения значений)
Object.keys() Возвращает массив собственных перечисляемых имен свойств объекта. Object.keys(obj); Объект (для извлечения ключей)
Массивы
Array.isArray() Проверяет, является ли объект массивом. Array.isArray([1, 2, 3]) → true Объект (для проверки)
Array.from() Создает новый массив из любого объекта, имеющего итерируемый интерфейс. Array.from('hello') → ['h', 'e', 'l', 'l', 'o'] Итерируемый объект (например, строка или Set)
Array.prototype.find() Возвращает первый элемент, удовлетворяющий условию в функции. [1, 2, 3].find(x => x > 1) → 2 Функция (принимает элемент массива и возвращает boolean)
Array.prototype.includes() Проверяет, содержит ли массив определенный элемент. [1, 2, 3].includes(2) → true Элемент (для проверки)
Array.prototype.some() Проверяет, удовлетворяет ли хотя бы один элемент условию. [1, 2, 3].some(x => x > 2) → true Функция (принимает элемент массива и возвращает boolean)
Array.prototype.every() Проверяет, удовлетворяют ли все элементы массиву условию. [1, 2, 3].every(x => x > 0) → true Функция (принимает элемент массива и возвращает boolean)
Array.prototype.push() Добавляет один или несколько элементов в конец массива. arr.push(4) Элементы (для добавления)
Array.prototype.pop() Удаляет последний элемент из массива. arr.pop() Нет
Array.prototype.shift() Удаляет первый элемент из массива. arr.shift() Нет
Array.prototype.unshift() Добавляет один или несколько элементов в начало массива. arr.unshift(0) Элементы (для добавления)
Методы строк
replace() Заменяет часть строки на другую. 'Hello world'.replace('world', 'JavaScript') → 'Hello JavaScript' Подстрока (для поиска) и замена (строка или регулярное выражение)
toLowerCase() Преобразует строку в нижний регистр. 'HELLO'.toLowerCase() → 'hello' Нет
toUpperCase() Преобразует строку в верхний регистр. 'hello'.toUpperCase() → 'HELLO' Нет
trim() Удаляет пробелы с начала и конца строки. ' Hello '.trim() → 'Hello' Нет
repeat() Повторяет строку несколько раз. 'a'.repeat(3) → 'aaa' Число (сколько раз повторить строку)
charAt() Возвращает символ по указанному индексу. 'hello'.charAt(1) → 'e' Индекс (для поиска)
split() Разделяет строку на массив подстрок. 'a,b,c'.split(',') → ['a', 'b', 'c'] Разделитель (строка или регулярное выражение)
Работа с числами
Number() Преобразует строку или другой тип в число. Number('123') → 123 Строка (для преобразования)
parseInt() Преобразует строку в целое число. parseInt('123.45') → 123 Строка (для преобразования)
parseFloat() Преобразует строку в число с плавающей точкой. parseFloat('123.45') → 123.45 Строка (для преобразования)
isNaN() Проверяет, является ли значение NaN. isNaN('123') → false Значение (для проверки)
Math.random() Генерирует случайное число от 0 до 1. Math.random() → 0.123456 Нет
Math.round() Округляет число до ближайшего целого. Math.round(4.5) → 5 Число (для округления)
Math.max() Находит наибольшее из чисел. Math.max(1, 2, 3) → 3 Несколько чисел
Math.min() Находит наименьшее из чисел. Math.min(1, 2, 3) → 1 Несколько чисел
Типы данных
typeof Оператор, который возвращает тип данных. typeof 'Hello' → 'string' Нет
instanceof Оператор, который проверяет, является ли объект экземпляром класса. arr instanceof Array → true Объект и класс (для проверки)
События
focus Событие, возникающее при получении фокуса на элементе. element.addEventListener('focus', function() { console.log('Focused'); }); Событие (объект события)
blur Событие, возникающее при потере фокуса на элементе. element.addEventListener('blur', function() { console.log('Blurred'); }); Событие (объект события)
change Событие, возникающее при изменении значения элемента формы. input.addEventListener('change', function() { console.log('Changed'); }); Событие (объект события)
input Событие, возникающее при изменении значения элемента формы. input.addEventListener('input', function() { console.log('Inputted'); }); Событие (объект события)
keydown Событие, возникающее при нажатии клавиши. document.addEventListener('keydown', function(event) { console.log(event.key); }); Событие (объект события)
click Событие, возникающее при клике на элемент. element.addEventListener('click', function() { console.log('Clicked'); }); Событие (объект события)
submit Событие, возникающее при отправке формы. form.addEventListener('submit', function(event) { event.preventDefault(); }); Событие (объект события)
Работа с DOM
getElementById() Возвращает элемент с указанным id. document.getElementById('myId'); id (строка)
querySelector() Возвращает первый элемент, соответствующий CSS-селектору. document.querySelector('.myClass'); CSS-селектор (строка)
querySelectorAll() Возвращает все элементы, соответствующие CSS-селектору. document.querySelectorAll('.myClass'); CSS-селектор (строка)
setAttribute() Устанавливает атрибут для элемента. element.setAttribute('src', 'image.jpg'); Имя атрибута и значение
Асинхронность/Promises
Promise Объект, представляющий результат асинхронной операции. let promise = new Promise((resolve, reject) => { resolve('Success'); }); Функция (для обработки результата или ошибки)
then() Метод для обработки успешного завершения промиса. promise.then(result => { console.log(result); }); Функция (для обработки результата)
catch() Метод для обработки ошибок промиса. promise.catch(error => { console.error(error); }); Функция (для обработки ошибки)
finally() Метод для выполнения кода после завершения промиса, независимо от успеха или ошибки. promise.finally(() => { console.log('Done'); }); Функция (для выполнения после завершения)
